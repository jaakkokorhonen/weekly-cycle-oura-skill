Pipeline Detail
pipeline.py — suoritusjärjestys
1
Fetch
oura_client.py hakee sleep · readiness · activity · heartrate. Kirjoittaa data/raw_inputs/YYYY-MM-DD.json (cache).
2
Normalize
Muunnetaan kaikki aikaleimät UTC+EEST. Yhdistetään events.jsonl-tiedot. Validoidaan skeema (#2).
3
Compute derived features
derived_hrv_delta_pct = (nightly_hrv − 14d_median) / 14d_median. derived_rhr_vs_30d, derived_active_kcal.
4
Classify
rule_engine.py: päiväluokka HIGH_LOAD / INTEGRATION / BASELINE + tilakoneen tila.
5
Recommend + Write
recommendation_engine.py tuottaa JSON-suosituksen. Kirjoitetaan data/records/ ja lisätään rivi data/runs.jsonl:ään.
